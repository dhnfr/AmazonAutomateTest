# AmazonAutomateTest

## Prerequisite
- Python >= 3.8
- pip
- Chrome Driver
- Clone this repository

## Intall requirement at virtual environment
- Create virtual environment
``` python3 -m venv venv ```
- Activate virtual environment
``` source venv/bin/activate ```
- Install requirements
``` pip install -r requirements.txt ```

## Run Test
``` robot <filename> ```
